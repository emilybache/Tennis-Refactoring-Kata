Tennis in Erlang
======================

You will need to install [erlang](https://www.erlang.org/), and [rebar3](https://github.com/erlang/rebar3). I recommend using JetBrains IntelliJ [Erlang plugin](https://www.jetbrains.com/help/idea/erlang.html).


Build
-----

    $ rebar3 compile

Run Tests
---------

    $ rebar3 eunit
